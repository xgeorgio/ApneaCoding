Παρουσίαση:<br/>
<b>«Προηγμένα συστήματα τεχνητής νοημοσύνης: Ώρα να μιλήσουμε για ανθρώπινα δικαιώματα και δημοκρατία»</b>

<p>Περιεχόμενα: Το σεμινάριο θα συζητήσει με ποιους τρόπους προηγμένα συστήματα τεχνητής νοημοσύνης έχουν δυσμενείς επιπτώσεις στην άσκηση ανθρωπίνων δικαιωμάτων ή/και σε θέματα δημοκρατικής συμμετοχής. Θα εστιάσουμε σε συστήματα τεχνητής νοημοσύνης υψηλού ρίσκου/κινδύνου και σε συστήματα απαγορευμένων πρακτικών σύμφωνα με τις ρυθμίσεις Ευρωπαϊκής Πράξης για την τεχνητή νοημοσύνη. Θα συζητήσουμε παραδείγματα, όπως χειριστικά συστήματα, συστήματα κοινωνικής βαθμολόγησης (social scoring), αναγνώρισης προσώπου σε πραγματικό χρόνο (real-time facial recognition), και συστήματα σε εκπαίδευση.</p>
